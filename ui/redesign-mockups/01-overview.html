confer · dashboard redesign

The fleet is a place, not a list.

Today's Overview ranks every alert into one flat column. But a fleet has structure — hubs you trust differently, agents that live on fixed machines, work that flows between them. This redesign makes that structure the interface: position tells you who and where, appearance tells you how they're doing, and the "what needs you" queue rides on top of the map instead of replacing it.

why this is better

Five moves, each fixing something the flat list gets wrong.

⬤ / ○

Appearance encodes state

A live agent breathes with a soft glow; a down one goes hollow and dim. Health is a gestalt you feel across the map — not a number you parse. Work Orbit being out reads instantly, before you read a word.

Astrolabos · position=identity, appearance=state
▚

Trust domains stay separate

Your internal hub gets a solid teal frame; a foreign hub gets a dashed magenta one and can never mint a top-priority alarm without local corroboration. The review's #1 finding — the queue flattened trust — designed away.

review BX4VP1 · trust-tier as first-class
◷

"All clear" can't be a stale lie

Every hub shows when it last synced, not just when the page last polled. A calm map backed by three-hour-old data now says so out loud — the zeroth signal of a triage view is whether you can trust the view.

review BX4VP1 · sync-freshness in the masthead
▲

Attention rides on the map

The "who + verb + copy-the-name" triage — the best part of today's design — is kept, but each item now points back to where it lives: a down node, a stuck request between two agents. You act, then you see the place you acted on.

design/47 · the good part, anchored
⌄

One honest glance, then depth

Level one is the whole fleet in a screen. A hub opens to its board; an agent opens to its threads. No fabricated "seen" dots, no synthesized rosters — every mark on this map is a real fact folded from the log.

Lacuna · layered complexity · empty-state contract
●

Work has a home

Requests aren't abstract rows — they sit inside the hub they belong to, colored by state: in-flight (cyan), stuck (amber ▲), plain. WIP shows on the agent carrying it, so "who's on what" is visible instead of derived.

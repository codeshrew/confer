why by machine, and living

Real topology, real pulse.

▟Organized by machine

Agents are processes on hosts. Grouping by machine is the honest topology — stable positions you learn, and a whole machine going dark reads at once (Hestia).

⬤Living presence

Live agents breathe; stale fade; down go hollow. Fleet health is a gestalt you feel across the deck before reading a word. (appearance = state)

◈Drill-in composes cards

Open an agent → its WIP is mini ticket cards (same system as the board), its pulse a real activity chart, its trust the real signing key. No new vocabulary.

▚Trust travels

Each agent shows its real key state; hub membership dots are tier-colored. A foreign agent reads foreign here too — law #2, on the crew deck.

↩The Overview's drill target

Clicking an agent on the Overview lands here. Fleet is the *inspect* layer for agents, the way the ticket card is for requests.

✓Honest by construction

Liveness, heartbeat, trust, WIP, host, hubs — all real projections. Activity sparklines bucket real message timestamps; flag to backend if it must serve them.

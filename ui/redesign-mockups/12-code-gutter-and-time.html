Three channels in the gutter, one timeline in the sidebar

Shape = scope

▤ file whole-file — lives in the top lane, never on a line
▐ range a bracket spanning the lines it's about
● line a tick — one line, one thought

You know the scope before you read a word — and the sidebar header mirrors it.

Color = meaning · column = overlap

▐ open question ▐ note ▐ needs owner ▐ blocked ▐ resolved

One hue, one meaning (our settled palette). Two threads on overlapping lines get side-by-side columns, so they never blur. Hover peeks; click focuses; j/k flows anchor-to-anchor. The minimap shows every cluster in the whole file, folded regions included.

Time = the spine

● this pinned to the version you're viewing
◷ older was about an earlier version — with ↳ align code to this

The sidebar lays a range's conversations oldest→newest so you see the discussion evolve. A drifted marker in the gutter warns when the lines have moved since. Snap the code to any conversation's exact commit — no doubt what was referenced.
